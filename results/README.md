# Results Folder
This folder contains processed outputs generated from reconstruction and noise analysis.
Current files:
- `Reconstructed_Conductivity_Data.csv`  
  Reconstructed conductivity values obtained from FEM-based EIT image reconstruction.
- `Noise_Summary_PerElectrode.csv`  
  Per-electrode statistical summary including mean voltage, standard deviation, signal-to-noise ratio (SNR), and coefficient of variation (CV).
