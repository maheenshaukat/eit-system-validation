# Results Folder
This folder contains processed outputs generated from reconstruction and noise analysis.
Current files:
- `Reconstructed_Conductivity_Data.csv`  
  FEM-based reconstructed conductivity values.
- `Noise_Summary_PerElectrode.csv`  
  Per-electrode statistical summary (mean, std, SNR, CV).
- `Reciprocity_Error.csv`  
  Matrix of |Vij - Vji| used for reciprocity validation.
- `Reciprocity_Error_Summary.csv`  
  Summary statistics of reciprocity error across electrodes.
- `SNR_Results.xlsx`  
  Signal-to-noise ratio results computed per electrode pair.
